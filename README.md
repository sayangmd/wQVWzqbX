# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行海滩垃圾检测
- 提供选择模型功能
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 支持选择单张图片或图片文件夹进行识别
- 支持选择视频文件进行识别
- 支持调用本地摄像头进行识别
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 支持选择检测模型

## 适用场景
- 海滩环境监测
- 海洋垃圾识别与分类
- 环保数据收集与分析
- 海滩清洁管理辅助

## 技术特点
- 基于YOLOv8目标检测算法
- 使用Python 3.9开发
- 集成OpenCV、PyQt5、Ultralytics和PyTorch1.9等库
- 包含900张标注数据集，包含1类垃圾（Sampah）
- 提供完整源码、训练好的模型、数据集和运行教程
- 支持PyCharm+Anaconda或VSCode+Anaconda开发环境
